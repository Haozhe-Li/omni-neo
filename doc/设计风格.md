# AI Agent 网页设计规范文档 (Perplexity 风格)

## 一、 设计核心哲学：Invisible Design (隐形设计)

- **目标**：界面应当是“信息的容器”，而非设计感的展示场。
- **关键词**：斯堪的纳维亚地铁感、专业、克制、事实驱动、透明。
- **核心准则**：如果一个设计元素不能帮助用户更好地阅读或搜索，就删掉它。

------

## 二、 视觉风格规范 (Visual Identity)

### 1. 色彩系统 (Palette)

避免使用高饱和度的“科技蓝”，转而使用带有温度的中性色。

- **底色 (Background)**：
  - 浅色模式：`#F3F3EE` (带有纸张质感的米白) 或纯白。
  - 深色模式：`#191A1A` (深灰黑，而非纯黑)。
- **文字色 (Typography)**：
  - 标题/正文：`#000000` (浅色模式) / `#FFFFFF` (深色模式)。
  - 辅助文本：使用 60% 透明度的灰，确保层级感。
- **强调色 (Accent)**：
  - 仅用于链接、引用源标号和主要按钮。推荐：`#20B2AA` (洗练的青色) 或沉稳的深蓝色。

### 2. 字体系统 (Typography)

- **品牌/标题字体**：选择具有“墨水陷阱 (Ink Traps)”细节的 Grotesk 字体（如 *FK Display* 或 *Inter*）。
- **正文/代码字体**：追求极致易读。
  - 推荐方案：`system-ui, -apple-system, sans-serif`。
- **字号层级**：
  - 输入框文本：24px - 32px (突出交互中心)。
  - 回答正文：16px (行高设为 1.6，增加阅读舒适度)。

### 3. Logo 与图标

- **Logo 风格**：全小写字母，无衬线，字间距略微紧凑。
- **图标规范**：使用细线条 (Thin/Light weight) 图标，线条末端圆润，整体保持空灵感。

------

## 三、 交互与布局规范 (UI/UX)

### 1. 结构布局

- **侧边栏 (Sidebar)**：
  - 默认可折叠。折叠后仅保留极简图标。
  - 使用毛玻璃效果 (Backdrop-filter) 或与底色微小的色差进行区分。
- **内容区 (Content)**：
  - **Max-width**: 限制在 800px - 900px 之间，防止长行文字导致阅读疲劳。
  - 居中对齐，留白充足。

### 2. 关键组件设计

- **搜索/输入框 (The Vessel)**：
  - 作为页面的视觉锚点。
  - 样式：大圆角矩形，微弱的边框或浅色阴影。
  - 交互：获取焦点时边框不应过度闪烁，使用平滑的过渡动画。
- **引用源卡片 (Sources)**：
  - 横向排列的小方卡片。
  - 包含：网站 Favicon、简短标题、序号。
- **回答生成动画**：
  - **打字机效果**：速度要适中，不要有过于生硬的跳动。
  - **加载状态**：模仿 Perplexity 的光标旋转（由光标演变为星号 Logo 的微动画）。

## 四、 开发实现要点 (Tech Implementation)

### 1. CSS 变量参考

CSS

```
:root {
  --bg-primary: #f3f3ee;
  --text-main: #1a1a1a;
  --accent-color: #005a5a;
  --border-subtle: rgba(0,0,0,0.05);
  --font-main: 'Inter', sans-serif;
}

/* 侧边栏模糊效果 */
.sidebar {
  backdrop-filter: blur(10px);
  background: rgba(243, 243, 238, 0.8);
}
```

### 2. 动画原则

- **Duration**: 所有的过渡时间控制在 `200ms` - `300ms`。
- **Easing**: 使用 `cubic-bezier(0.4, 0, 0.2, 1)` 实现自然物理感的启停。
